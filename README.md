## 🙂 간략소개
SAP ERP 운영 경험을 기반으로, 현업의 반복 업무를 AI로 구조화하고 실제 서비스로 구현하는 IT 기획자입니다.

IT 컨설팅·SI 환경에서 다수 고객사의 SAP ERP(FI) 운영·유지보수를 담당하며, 현업 사용자의 반복 이슈를 분석하고 이를 기능 개선으로 연결하는 경험을 쌓았습니다.  
이 과정에서 단순 개발이 아닌 **안정적인 운영과 지속적인 개선을 전제로 한 기획 역량(요구사항 정의, 예외 처리 설계, 사용자 중심 문제 해결)**을 체득했습니다.

이후 AI 기반 서비스 2건을 **기획부터 개발·배포·운영까지 End-to-End로 전담 수행**하며, 실제 업무 흐름을 반영한 서비스 설계와 빠른 실행력을 검증했습니다.  
특히 Claude Code(AI 페어 프로그래밍)를 활용해 기획자 관점에서도 구현–검증–개선 사이클을 단축했습니다.
<br>

- **AI 법률 사건관리 SaaS(SLA)** — 송무 직원 설문조사 기반 업무 프로세스 구조화, AI 문서 자동 분류·요약, 150+ REST API, 49개 페이지, AWS 배포 (4주)
- **법무법인 웹사이트 리뉴얼** — 경쟁사 UI/UX 분석 → v1→v33 반복 개선, 데이터 크롤링 자동화, 반응형 웹 구축 (8주)

<br>
이외에도 ML, DL, LLM 기반 AI 프로젝트를 수행하며, 모델 선정부터 UI/UX 설계, 예외 처리, 배포까지 전 과정을 경험했습니다.

**SAP 기반 도메인 이해도와 AI 서비스 구축 경험을 바탕으로,  
현업의 문제를 실제 운영 가능한 서비스로 구현하는 IT 기획자로 기여하고자 합니다.**

## 🎯 핵심 역량

**AI 서비스 기획 및 사용자 중심 개선** <br>
송무 직원 설문조사 → 경쟁사(로탑/YK/대륜) 기능 비교 분석 → 기능 우선순위 도출 → v1→v33 반복 개선까지, 사용자 니즈에서 출발하여 AI 서비스를 기획하고 지속적으로 개선한 실무 경험

**풀스택 웹 서비스 구축** <br>
AI 서비스 2건을 기획·설계부터 AWS 배포까지 전 과정 직접 수행. 프로덕션 레벨 시스템 구축 경험을 바탕으로 개발팀과 기술적 맥락에서 협업 가능한 기획 역량 보유 (Claude Code 활용)

**기업 시스템(SAP) 유지보수 및 기능 개선** <br>
다수 고객사의 SAP ERP(FI) 시스템을 담당하며 현업 요청 대응, 오류 분석, 기능 개선을 수행. 반복 이슈의 원인을 구조적으로 분석하고 기능 개선으로 연결한 경험이 AI 서비스 기획에서 요구사항 정의·예외 처리 설계 역량으로 이어짐

**클라우드 인프라 구축 및 배포** <br>
AWS EC2 + RDS + S3 + PM2 + Nginx + SSL 풀스택 배포 파이프라인 직접 구축 (프로젝트 2건 동일 구성). 서비스 안정성·데이터 정합성을 우선하는 설계 관점 보유

**실사용 환경 중심의 AI·데이터 설계** <br>
머신러닝·딥러닝·LLM 프로젝트에서 예외 상황, 데이터 누락, 입력 오류 등을 고려해 모델과 시스템을 설계하며 실사용 환경 기준의 예외 처리 및 서비스 설계 수행

### 2. ⚖️ SLA (Smart Legal Assistant) — AI 기반 법률 사건관리 SaaS
"보관을 넘어 분석으로, 기록을 넘어 지능으로 — 송무 직원의 업무 흐름을 완벽히 이해한 법률 사무 자동화 플랫폼"

| **규모** | 프론트엔드 49개 페이지, 71개 컴포넌트, 백엔드 127개 API, DB 20개 테이블 |
| **기술 스택** | Frontend: React 19, Vite 7, Tailwind CSS 4, FullCalendar / Backend: Express.js 5, MySQL (AWS RDS), JWT 인증 / AI: Anthropic Claude Haiku 4.5 / Storage: AWS S3 / DevOps: AWS EC2, PM2, Nginx, GitHub |
| **주요 기능** | 사건 관리, 기일 캘린더, AI 문서 분석·요약, 비용 결재, 법률 계산기 10종, 고객 CRM, 내부 채팅, 의뢰인 포털 |



**🎯 핵심 성과**
- 🤖 **AI 문서 분석 시스템**: Anthropic Claude 연동으로 법률 문서 자동 분류·요약·태깅 구현 (드래그앤드롭 업로드 → AI 분석 → 카테고리 분류)
- 📊 **150+ REST API 설계**: 사건/기일/문서/비용/고객/결재/알림 등 법률 사무 전 영역을 커버하는 백엔드 아키텍처
- 🏗️ **20테이블 관계형 DB 설계**: 사건-의뢰인 N:N 관계, 7단계 비용 결재 워크플로우, 활동 감사 로그 등 실무 요구사항 반영
- 🔒 **엔터프라이즈급 보안**: JWT 인증 + 토큰 버저닝 + Rate Limiting + Helmet CSP + AES-256-GCM 암호화
- ☁️ **AWS 풀스택 배포**: EC2(서버) + RDS(DB) + S3(파일) + PM2 + Nginx + Let's Encrypt SSL



**💡 주요 기술 경험**
- React 19 + Vite 7 SPA 아키텍처: Context API 기반 상태 관리 (Auth/Preferences/Office), 49개 페이지 라우팅
- Express.js REST API 설계: 127개 엔드포인트, 미들웨어 체인 (helmet → compression → CORS → rate-limit → auth)
- MySQL 스키마 설계: 20개 테이블, FULLTEXT 인덱스, ENUM 타입, FK CASCADE, 마이그레이션 v1→v2
- AI/LLM 연동: Anthropic Claude API로 문서 분석·초안 작성·판례 검색·사건 요약·결과 예측 5개 AI 기능 구현
- AWS S3 Presigned URL: 클라이언트 직접 업로드 방식으로 서버 부하 제거
- 법률 도메인 계산기: 인지대·송달료·손해배상·소멸시효 등 10종 법률 계산 로직 구현
- PWA 지원: Service Worker + Web Push Notification + 오프라인 페이지
- 7단계 비용 결재 워크플로우: 신청→검토→승인→집행→정산→완료→반려 상태 머신 설계<br>
